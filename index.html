O vortex de informação organizado em um banco de dados

Antes de qualquer análise, toda a informação caótica publicada na internet é organizada em um banco de dados.

Esta coleta é possibilitada por um software construído para este propósito. Seu papel é coletar informação da internet e estruturá-la.

O banco de dados pode crescer indefinidamente. Tudo depende da quantidade de informação necessária para a análise.

Caso seja necessário saber mais sobre uma recorrência identificada, basta fazer o programa coletar mais informação.

Ou, que tal relatórios periódicos sobre os dados coletados? É possível analisar informação resultante de interações online à medida em que acontecem.
